
# WEB API EXERCISE

A simple CRUD web app build with React, Vite, Axios to interactive with API


## How to run

First, clone this repos

```bash
  git clone https://github.com/daokiencuong/Web-API-Exercise
```

After that, go to the source code folder, install dependencies for this project

```bash
  npm i
```

Final, run this project in dev mode with

```bash
  npm run dev
```

Go to http://localhost:5173 to view project
